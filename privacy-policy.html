Privacy Policy

Last updated: July 20, 2025

1. Information Collected

This bot temporarily processes user IDs, server IDs, command messages, and channel IDs solely for executing commands.

2. Data Storage

No personal data is permanently stored. All processing is done in memory.

3. Logging

The bot may log command usage for debugging or moderation purposes. These logs are not public.

4. Third-Party Services

This bot operates within Discord’s infrastructure. Use is subject to Discord’s Privacy Policy.

5. Your Rights

If any data is stored (e.g., for settings), you may request removal by contacting the bot owner.

For questions or concerns, contact the bot owner via Discord.
